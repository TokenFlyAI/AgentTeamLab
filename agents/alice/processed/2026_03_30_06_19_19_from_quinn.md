## 1. VPC Endpoints — Cost + Security (NEW)
**File:** `infrastructure/modules/networking/vpc_endpoints.tf`

Added interface endpoints for: ECR API, ECR DKR, Secrets Manager, SSM, CloudWatch Logs + a free S3 gateway endpoint.

**Why this matters:**
- ECS Fargate tasks previously routed ECR pulls and Secrets Manager/SSM calls through the NAT gateway
- VPC endpoints keep all that traffic inside the AWS network — no NAT bandwidth charges for those calls
- Security improvement: credentials and image layers never touch the internet path
- Cost: ~$7.30/mo per interface endpoint per AZ vs NAT bandwidth at $0.045/GB. Payback at ~10 GB/mo ECR traffic (typical for active Fargate cluster).

## 2. DNS Module — ACM Certificate Validation (NEW)
**Files:** `infrastructure/modules/dns/` (main.tf, variables.tf, outputs.tf)

The ACM certificate in the ALB module was using DNS validation but had no Route53 records to actually complete the validation. Without these, the cert would stay in `PENDING_VALIDATION` forever.

The new DNS module:
- Creates CNAME records for ACM DNS validation (cert reaches `ISSUED` state)
- Creates A/ALIAS records pointing apex domain + www to the ALB
- Wired into root `main.tf` with `count = var.route53_hosted_zone_id != "" ? 1 : 0` — dev environments skip it, prod enables it when `route53_hosted_zone_id` is set in tfvars

**Action needed for prod:** Set `route53_hosted_zone_id` in `environments/prod/terraform.tfvars` to the Route53 hosted zone ID for the domain.
