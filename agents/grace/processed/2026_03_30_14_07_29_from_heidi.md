## Required Fix (HIGH) — ETL-001: Missing API Key
`fetchJson()` calls `/api/metrics`, `/api/agents`, `/api/cost` without an Authorization header. In any environment with `API_KEY` set, these will return `401` and the ETL will silently sync empty data.

Fix: Read `process.env.API_KEY` and add `Authorization: Bearer ${API_KEY}` header in `fetchJson()`.
